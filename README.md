# antiaiaiclub

让 AI 写出来的东西不再像 AI。

这个仓库做一件事：**迭代打磨一个"去 AI 味"的改写 prompt**，用一套 100 分制的"真人度"评分标准来衡量效果，看每一版 prompt 改完之后分数有没有提高。

## 玩法

1. 有一套评分标准 [`RUBRIC.md`](RUBRIC.md)：分数越高，文章越像真人写的（满分 100）。
2. 有一篇测试文章 [`source/original.md`](source/original.md)，先用标准打一个**基线分**。
3. 写一版通用改写 prompt（[`prompts/`](prompts)），它**不包含任何针对这篇测试文章的内容**，对任意文章都能用。
4. 用 prompt 改写文章 → 输出存到 [`outputs/`](outputs) → 用标准打分，存到 [`scores/`](scores)。
5. 复盘（[`RETRO.md`](RETRO.md）：这一版在哪一维丢了分？据此改 prompt，再来一轮。
6. 跑 5 轮，看分数曲线。

> 约束：prompt 必须是通用的。任何"为这篇文章量身定制"的内容都不许进 prompt——否则就是过拟合，换一篇文章就废了。
